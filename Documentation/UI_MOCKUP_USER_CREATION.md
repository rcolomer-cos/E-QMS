## Color Scheme

### Primary Colors
- **Primary Blue:** #3498db (buttons, accents)
- **Hover Blue:** #2980b9 (button hover state)
- **Success Green:** #2ecc71 (status indicators)
- **Error Red:** #dc2626 (validation errors, error text)
- **Warning Yellow:** #fef3c7 (warning box background)

### Neutral Colors
- **White:** #ffffff (backgrounds, cards)
- **Light Gray:** #f9fafb (section backgrounds)
- **Medium Gray:** #6b7280 (secondary text)
- **Dark Gray:** #111827 (primary text)
- **Border Gray:** #e5e7eb (borders, dividers)

### Role Badge Colors
- **Admin:** Red (#e74c3c)
- **Manager:** Purple (#9b59b6)
- **Auditor:** Blue (#3498db)
- **User:** Green (#2ecc71)
- **Viewer:** Gray (#95a5a6)

### Group Badge Style
- **Background:** Light gray (#f0f0f0)
- **Text:** Dark gray (#555)
- **Size:** Small, compact

## Responsive Behavior

### Desktop (> 768px)
- Dialog width: 700px max
- Two-column layout for First/Last Name and Phone/Department
- Full table with all columns visible

### Tablet (768px)
- Dialog width: 90% of viewport
- Single-column form layout
- Scrollable table

### Mobile (< 768px)
- Dialog width: 95% of viewport
- Stacked form fields
- Generate button full-width below password field
- Simplified table view or card layout

---

## Interactions

### Hover States
- Buttons darken on hover
- Table rows highlight on hover
- Checkboxes show focus ring

### Loading States
- Create User button shows "Creating..." during submission
- Disabled state for all inputs during submission

### Success States
- Toast notification appears after successful creation
- Credentials dialog replaces create dialog

### Error States
- Inline error messages below fields
- Red borders on invalid fields
- Toast notification for server errors

---

## Accessibility Features

- **ARIA Labels:** All form fields properly labeled
- **Keyboard Navigation:** Tab through all fields
- **Focus Indicators:** Visible focus rings on all interactive elements
- **Error Announcements:** Screen reader friendly error messages
- **Semantic HTML:** Proper heading hierarchy and landmarks
- **Required Fields:** Marked with asterisk and aria-required
